## Cocôzap // [DataLabe](http://datalabe.org/)

Mobilização, Mapeamento E Incidência Para Saneamento Básico Em Favelas

Build with React, Gatsby.js and Node.js

### Installation

- Install [Gatsby.js](https://github.com/gatsbyjs/gatsby)

- Clone the project

- Scripts:

``` bash
# development
gatsby develop

# build application
gatsby build
```
